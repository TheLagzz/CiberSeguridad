# ¿Qué es OSINT?
Las siglas OSINT es Open Source Intelligence, se le llama OSINT a una serie de procesos que tienen como objetivo hacer uso de fuentes de información publica o abiertas para así poder buscar y recopilar toda la información posible sobre un objetivo en concreto ya se por ejemplo una persona o una empresa con el fin de recopilar toda esta información y crear inteligencia a partir de ella, mediante la interconexión de puntos, viendo patrones repetitivos y mas cosas. El OSINT no es solo para ciberseguridad, se puede utilizar en muchas otras áreas. El OSINT es completamente legal ya que como tal no estamos interactuando con ningún objetivo porque solo estamos consultando información pública que es accesible para cualquier persona y por ende es legal.

# ¿Cuáles son las fuentes abiertas donde se consigue información?
Básicamente cualquier lugar que tenga información pública y para ellos se pueden utilizar varios métodos como los motores de búsqueda como Google o el uso de redes sociales, hay maneras de hacerlo manualmente pero también hay manera de automatizarlo un poco con herramientas, por ejemplo a la herramienta le das el nombre de una entidad y esta se va a encargar de buscar toda la información que pueda conseguir sobre dicha entidad o persona. El OSINT para la certificación eJPT no es muy relevante como para el examen pero es importante saberlo porque en muchas partes se podrían utilizar.
Las herramientas públicas para OSINT no sirven para mucho porque no destacan mucho mas allá de saber si un nombre de usuario existe en una red social, las verdaderas herramientas de OSINT no están públicas y suelen ser mas potentes de lo que la gente piensa.

# OSINT Framework
No es una herramienta, se puede definir mas como un repositorio que almacena de forma ordenada por categorías varios recursos para OSINT (aunque hay herramientas que no son de reconocimiento pasivo) y la recopilación que te hace es dándote los hiperenlaces a las herramientas en cuestión o meramente a los Dorks en buscadores que ya están previamente configurados. Este framework ofrece múltiples herramientas que son conocidas o básicas.

# Pentester.com
Es una plataforma que te ayuda a revelar que datos se han filtrado en la red.
# OSINT Industries
Es parecido a Pentester.com, pero OSINT Industries se enfoca mas en ver los datos filtrados. Estos sitios web te ayudan a hacer OSINT te ayuda a conseguir patrones repetitivos. OSINT Industries no es de uso gratuito pero es muy potente por eso es de paga.

# Sherlock OS
OSINT es aprovecharnos de muchas herramientas y luego a partir de la información que encontramos poder crear inteligencia viendo patrones repetitivos y demás. Hay muchísimas herramientas que se pueden utilizar para OSINT pero hay un sistema operativo que se enfoca en OSINT que es llamado **Sherlock OS**.

# Navegador VS Buscador
Un navegador es un software que esta enfocado en interpretar hipertexto y de esta forma permitirle al usuario visualizar paginas web de un sitio y un buscador es un software que permite encontrar información alojada en cualquier parte de la red información a la que ha accedido previamente ese buscador y la ha indexado con base a sus criterios.
