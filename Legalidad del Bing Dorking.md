 Es lo mismo o casi lo mismo pero con el buscador Bing que es de Microsoft, los operadores y eso van a funcionar casi que por seguro. Pero tiene unos extras o diferentes:
### "loc:1"
Si quisieramos ver paginas de un país distinto deberíamos escribir "loc:ES" para España o para Estados Unidos "loc:US".

### "feed:"
 Se puede usar para buscar feeds de los dominios como "feed:microsoft.com".

### "ip:"
Se solía usar mucho Bing por este operador para buscar la dirección ip del host pero ya esta prácticamente obsoleto.

El operador "inurl:" no sirve en este buscador.

Seguido del sitio web accedes directo a el, como "\\Wikipedia" vas directo a la pagina de Wikipedia

# Legalidad de Yandex
Se suele usar Yandex para la búsqueda inversa de imágenes no como para información en si. Existe "url:" que es el "site:" en Google. Existe "mime:" que es como el filetype de Google. Existe "lang:" que muestra paginas segun el leguaje indexado como "lang:es ciberseguridad". "date:" sirve para filtrar por fechas y se pueden usar varios formatos
